# Tilburg University Student Preference Portal

A simple application for students to manage their interface theme settings.

## Features
- **Input**: Enter student name or ID.
- **Switch**: Toggle between Light and Marine (Dark) mode.
- **Visual Feedback**: Real-time color shift using Tilburg University brand colors.
- **Simulated Backend**: Flask API with artificial delay to demonstrate network activity.

## Local Development
1. Clone the repository.
2. Run `docker compose up --build`.
3. Access the frontend at `http://localhost:3000`.
4. The API is available at `http://localhost:5000/api/health`.

## Brand Guidelines Applied
- **Primary Color**: Marine (#0c2340)
- **Accent Color**: Brons (#b46b2a)
- **Typography**: Arial/Helvetica
- **Tone**: Academic, clean, and professional.
